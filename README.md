# ✦ Lumina - Image Gallery

**Lumina** is a curated, responsive image gallery featuring dynamic category filtering and an immersive lightbox experience. Built with pure HTML, CSS, and JavaScript - no frameworks, no dependencies.


---

## ✨ Features

- 🖼️ **Dynamic Gallery Grid** - Responsive masonry-like layout
- 🔍 **Category Filtering** - Filter by Nature, Architecture, Abstract, or Travel
- 💡 **Fullscreen Lightbox** - Click any image for an elegant modal view
- ⌨️ **Keyboard Navigation** - Arrow keys to navigate, `ESC` to close
- 📱 **Fully Responsive** - Optimized for desktop, tablet, and mobile
- ♿ **Accessibility Ready** - ARIA labels, semantic HTML, keyboard focus management
- 🎨 **Aesthetic Design** - Custom fonts (Cormorant Garamond + Jost), smooth transitions

---


## 🛠️ Built With

- **HTML5** - Semantic structure
- **CSS3** - Grid, Flexbox, custom properties, animations
- **Vanilla JavaScript** - DOM manipulation, event handling, dynamic rendering
